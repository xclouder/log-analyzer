# UE时间戳格式支持更新

## 🎯 更新内容

已添加对 **Unreal Engine (UE)** 日志时间戳格式的完整支持！

## 📝 日志格式

UE日志中的时间戳格式：
```
[2025.11.04-19.19.39:123]LogTemp: Your log message here
```

## ✨ 用户输入格式

现在你可以使用简化的格式输入时间戳：

### 完整格式
```
11.04-19.19.39
```
- 格式：`MM.DD-HH.mm.ss`
- 示例：11月4日 19点19分39秒

### 简化格式（推荐）
```
11.04-19.19
```
- 格式：`MM.DD-HH.mm`
- 示例：11月4日 19点19分
- 秒会自动设置为00

## 🐛 修复的Bug

1. **换行符处理**
   - 问题：使用 `split('\n')` 在Windows上可能不正确
   - 修复：使用 `split(/\r?\n/)` 兼容所有平台

2. **时间戳格式不匹配**
   - 问题：UE格式无法识别
   - 修复：添加UE格式的正则表达式匹配

3. **用户输入复杂**
   - 问题：需要输入完整的年份和秒
   - 修复：自动补全年份，秒可省略

### 输入格式对照

| 你想查找的时间 | 输入格式 | 说明 |

3. **分段读取**：对于超大文件，按时间段分批读取
4. **结合过滤器**：读取后用过滤器快速找到错误
